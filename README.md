### How to run

```bash
$ sudo npm install -g ionic cordova
$ sudo npm install
```

Then, to run it, on IOS run:

```bash
$ ionic cordova platform add ios
$ ionic cordova run ios
```

To run it on Android just run:

```bash
$ ionic cordova run android
```
